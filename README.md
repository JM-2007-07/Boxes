# Electrical Services Landing Page

Современный лендинг для компании, занимающейся электромонтажными работами. Дизайн выполнен в Figma и предназначен для презентации услуг, преимуществ компании и удобной связи с клиентами.

## О проекте

Данный макет разработан для сайта электромонтажной компании. Основная задача сайта — показать услуги, повысить доверие клиентов и упростить подачу заявок на выполнение работ.

## Структура сайта

### Главный экран
Первый экран с основным заголовком, кратким описанием услуг и кнопкой для связи или оформления заявки.

### О компании
Секция с информацией о компании, опыте работы и профессиональном подходе к выполнению электромонтажных услуг.

### Услуги
Блок с основными направлениями работы:
электромонтаж, подключение оборудования, ремонт электрики, установка освещения и другие услуги.

### Преимущества
Секция с преимуществами компании: качество работы, безопасность, опыт специалистов и соблюдение сроков.

### Проекты
Блок с примерами выполненных работ или фотографиями объектов.

### Отзывы
Отзывы клиентов о качестве услуг и работе специалистов.

### Контакты
Информация для связи с компанией: телефон, адрес, электронная почта и форма обратной связи.

### Footer
Нижняя часть сайта с дополнительной навигацией и контактной информацией.
